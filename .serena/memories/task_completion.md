# Task Completion Checklist

When a task is completed in this project, perform the following steps:

1. **Format Code**: Run `go fmt ./...` to ensure consistent formatting.
2. **Lint Code**: Run `go vet ./...` to check for suspicious constructs.
3. **Run Tests**: Ensure all tests pass with `go test -v ./...`.
4. **Documentation**: Update `README.md` or other docs if the public interface or features change.
5. **Clean Logs**: Ensure no emojis or debug logs are left in the production paths.
